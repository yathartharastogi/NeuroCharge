# FRONTEND SPECIFICATION DOCUMENT
**Project:** NeuroCharge

## 1. DESIGN PRINCIPLES
**Style:**
- Clean
- Professional
- Modern
- Data-focused

**Key Integrations:**
- Aceternity UI components for sophisticated and smooth interactions

**Avoid:**
- Glassmorphism
- Neon effects
- Excessive animations

## 2. COLOR SYSTEM
**Light Mode:**
- Warm off-white backgrounds (#f7f3ee)
- Neutral grays
- Minimal accent colors

**Dark Mode:**
- Deep charcoal
- High readability

## 3. PAGES (CONSOLIDATED)
**Main Dashboard / Real-Time Monitor**
*Displays:* Battery Health, State of Charge, Temperature, Lifespan Prediction, Live Event Encoder Spikes, Thermal Alerts.

**Predictive Analytics & Digital Twin**
*Displays:* SOH trends, Capacity degradation over time, Temperature history, Future simulation toggles.

**Smart Recommendations & AI Explorer**
*Displays:* Fast charging frequency, Recommended charging levels, Potential lifespan gains, Stress level, Risk score, Prediction confidence.

## 4. UI COMPONENTS
- Navbar
- Sidebar
- Metric Cards
- Line Charts
- Recommendation Cards
- Alert Feed
- Data Tables

## 5. RESPONSIVENESS
**Supported:**
- Desktop
- Tablet
- Mobile

## 6. PERFORMANCE TARGETS
- Initial load < 2s
- API updates < 500ms
- Smooth chart rendering

## 7. ACCESSIBILITY
- Keyboard navigation
- High contrast support
- Screen reader compatibility
